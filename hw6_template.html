Part 2: Ratios and Categories Census Data

I used the ratio of aggregate number of minutes counties' populations spent commuting to work over each counties' total population in 2020. This normalization accounts for geographic areas of high population density, allowing us a better understanding of commute time patterns across the state. Each mode of data categorization provides a different view of the data, some clearly more helpful than others. For example, the first mode pictured, Equal Interval, provides the most useful view of the data. The equal intervals, each 1.6 units apart, begin at 7 and end at the value 15, and color categories are represented on the map. Also informative, but slightly less so, is the Natural Breaks mode. In this map, the data is grouped logically based on values that fall close together. A con of this mode when applied to this particular dataset is that the first color category goes entirely unrepresented because it corresponds to the 0-0 range. In the case of this dataset, the least helpful mode was Equal Count (Quantile). Only the last three color categories were represented on the map because the first two corresponded with ranges in which no values fell within: 0-0 and 0-2.8
